    NFT会成为打开Web3的钥匙吗

    5/1/2022 Web3区块链NFT

    什么样的NFT才是值钱的

    #
    什么是NFT

    不可替代的代币 (NFT) 常常视为资产所有权的数字契约或演出门票。

    作为一种数字代币，NFT 对与资产相关的信息进行编码和加密保护，可能包括身份、所有权、资产历史、转售版税等数据。

    所有者可以在Mintable和OpenSea等平台上“铸造”NFT，这些平台也可作为 NFT 市场。代币被写入区块链，通常是以太坊，尽管较小的 NFT 友好区块链技术比比皆是。区块链为这些代币提供了一个安全的、全球分布式的分类账本，以便可以通过用户的区块链地址而不是姓名来追踪交易，如果所有者愿意，可以维护隐私。这种匿名或假名通常在虚拟世界受到重视。

    基于以太坊的 NFT 使用以太坊的***ERC-721***开放标准。这些代币是真正独一无二的；永远只会有一个与每个相关联的资产。相比之下，ERC-20代币是可替代的，尽管供应量通常有限。（想想音乐会门票，虽然 ERC-20 代币也可以用于可兑换的代金券、治理份额或社会社区建设，如音乐艺术家 RAC 的$RAC 代币。）在这两种情况下，稀缺性都是一个关键的价值驱动因素，但这对于 NFT 尤其如此。

    NFT到现在，只发展了九年。

    # NFT 作为视觉艺术

    “NFT 解放了艺术。传统艺术是有时间和空间的。你必须在正确的城市，去博物馆，被邀请去某人的家，等等。任何人，任何地方只要有互联网连接，就可以查看 NFT 并接收它们。这是一个巨大的突破。” ——Cameron Winklevoss，企业家兼 Gemini 交易所联合创始人

    从 2017 年 6 月推出CryptoPunks开始，视觉艺术与 CryptoKitties（上图）一起将 NFT 带入了主流意识，CryptoPunks包含 10,000 个独特的 24x24 像素艺术头像。朋克最初是免费提供给以太坊社区的，并帮助激发了 ERC-721 代币技术。

    CryptoPunks 系列发布后不久，便催生了二级转售市场。根据创作者 Larva Labs 的说法，“去年出售的朋克总价值为 199,519.76 ETH（404,723,844.09 美元）。” 迄今为止最大的朋克拍卖是 2021 年 3 月的#7804，成交价高达 757 万美元。朋克通常被视为社区和/或地位的象征，这可以解释说唱歌手Jay-Z 可能以大约 119,000 美元的价格购买#6095。

    NFT 包含智能合约，这是一种在给定条件下执行一组操作的技术。换句话说，如果一件事发生，那么另一件事就会执行。智能合约功能是 NFT 在视觉艺术中使用的核心，因为它为艺术家提供了一个新的收入渠道，而以往大部分收入会流入中介和转售。通过 NFT 智能合约，艺术家可以直接向买家出售一件作品（区块链确认的不可替代性确保买家拥有独特的所有权），并获得所有后续转售的版税百分比。如果 NFT 更改所有权，那么该交易的 X% 将转到艺术家的区块链钱包。智能合约自动处理这些操作，中间没有人参与或引入延迟。

    1、艺术品通常通过Foundation.app等市场出售。

    2、备受瞩目的作品可能会通过传统拍卖商进行，正如 Beeple在 2021 年 3 月通过佳士得打破记录的6930万美元 NFT拍卖所证明的那样。

    总而言之，CryptoArt 数据显示超过 700,000 件迄今为止，NFT 艺术品已售出超过 300,000 ETH（目前超过 6 亿美元）。

    NFT炒作是否消退？或许这样的说法还为时尚早。

    # 游戏中的NFT

    前面提到的 CryptoKitties 游戏于 2017 年在以太坊区块链上首次亮相。2019 年，CryptoKitties 团队宣布创建了一个名为Flow的新区块链，此后吸引了许多 NFT 发行商。

    当今最热门的 NFT 驱动游戏之一是Axie Infinity，就像 CryptoKitties 遇到 Pokémon Go，完成与怪物的战斗。

    该游戏的基于 ERC-20 的 Small Love Potion ($SLP) 代币可用作培育 Axies 的货币，并在全球交易所（例如币安）上进行游戏外交易（请参阅下面的 90 天价格图表）。Axie Infinity 现在每天有超过 40,000 名活跃用户，NFT 交易量为每月 1500 万美元，市值为 2B 美元，合作伙伴包括 Ubisoft、Binance 和三星。Axie Infinity 玩家可以通过 SLP 农业每月赚取数百美元。将 NFT 和区块链技术注入游戏领域，随时准备开辟在线娱乐的新阵线，并使玩赚钱游戏成为一种全球现象。有些人怀疑它甚至会取代当今非常流行的免费游戏模式。

    90 天 SLP/USD 价格图表。请注意 5 月价格下跌与广泛的加密货币市场一致，然后随着游戏的病毒式传播，价格在 7 月初迅速翻了三倍。

    从这里开始，将玩游戏带入元宇宙只是一小步。除了简单的游戏之外，我们可以很容易地想象，跨平台技术可以让基于 NFT 的角色或游戏中的物品出现在元宇宙用户的虚拟资产中。想要下一代无聊猿作为您的虚拟世界管家吗？它可能发生。

    # 音乐和写作中的NFT

    音乐、推文等等都可以被铸造成NFT。

    Mirror 及其 $WRITE 代币则吸引力更高，后者使用其代币进行投票/治理以及作家补偿. 同时，每篇 Mirror 文章都被铸造为与可交易的 $ESSAY 代币相关的 NFT。 读者用以太坊货币 (ETH) 购买 $ESSAY，从而拥有文章的一部分。多亏了智能合约，这使得论文产生的任何未来收入都可以与 $ESSAY 持有者分享，就像股票红利一样。

    Mirror 项目似乎它承载着将作者和读者联系起来并去除版税、中间人的有趣承诺。

    但Mirror文章的防盗是一个大问题（就像起点一直都没解决小说盗版的问题）

    # NFT 的乐趣和可收集性

    可收藏性一直是关于社会相关性和影响的感知价值和集体评估。

    篮球在许多人的生活中扮演着重要的角色，因此同样由 Dapper Labs（CryptoKitties 背后的公司）创建的 NBA Top Shot 成为 NFT 中最知名的品牌之一也就不足为奇了。用户购买和交易称为“时刻”的视频集锦作为 NFT。到 5 月底，该服务已突破百万用户大关，并在不到一年的时间内积累了超过 7 亿美元的销售额，其中大部分是 NFT 转售。

    还有什么火星之家、虚拟球鞋......尤其是当数字时尚与物理世界交叉时，NFT 服装正迅速流行起来。

    # NFT遇到房地产（这太不靠谱了，投机居多）

    虚拟世界是根据规则创建的，包括存在多少“土地”以及该土地在世界中的位置。从理论上讲，人们可以改变规则并创造更多的土地出售。

    在实践中，这将违反用户对土地稀缺性的看法，从而影响其价值，并可能导致该虚拟世界的房地产经济崩溃。虚拟世界里的经济蓬勃发展的部分原因是这种稀缺性，因此规则制定者不会恶意改变规则。

    一个合理完备的体系是必要的（就像国家的概念，而在web3时代，每个组织都要拥有一定范围内的解释权，那么监管是否是必要的，还是看不见的手自己调控？）

    房地产可能只是 NFT 侵入传统金融和投资领域的开始。期待看到 NFT 与去中心化金融 (DeFi) 桥接的多种方式，例如NFTfi，其中 NFT 可以作为借贷流动性池中的抵押品。正如备受瞩目的 NFT 拍卖可能是 NFT 拍卖的先兆一样，NFTfi 也是如此。

    在这两种情况和无数其他情况下，随着全球对 NFT 的访问和竞争的升温，实时交易性能将变得越来越重要。网络将面临越来越大的压力，以保持低延迟和无拥塞的交易。

    # 如果不相信的话，看开点，NFT才刚开始呢

    毕竟，我们仍处于 NFT 及其用例的开始阶段。IBM 正在努力将其专利转化为 NFT，以“帮助使专利更容易销售、交易、商业化或以其他方式货币化”。科学家们预计将某些个人的基因组作为 NFT 出售，即使只是为了筹款。NFT 越来越多地在去中心化金融 (DeFi) 中发挥作用，协助管理借贷。NFT 还可以帮助现实世界的库存跟踪和物流。

    如果这些 NFT 的好处可以轻松融入消费者的元空间生活，那么用户为什么不接受它们呢？

    形成共识的重要性，以及个人主义的展示。

    #
    无聊猿NFT是如何商业化的

    BAYC（Bored Ape Yacht Club）是2021年4月推出的NFT（数字藏品）项目，包括一万个独特风格的猿猴Bored Ape，每个猿猴的着装、造型都不尽相同，无聊猿NFT可以作为游艇俱乐部的会员资格。初推出时，无聊猿的单个售价仅为185美元，发布后即刻售罄，并引发了大量的明星追捧。

    # Bored Ape怎么火起来的？商业模式重点在哪？

    Yacht Club第一批NFT营收超过200万美元，而后Bored Ape的交易额超过了一亿美元，Bored Ape项目的成功，激发了市场对NFT追捧，之后市场推出了各式各样的NFT，加密粉丝和收藏爱好者可以根据自己的爱好，选择购买各类不同的NFT。

    从主要宣传场所来看，NFT的主要宣传场所在Twitter，各类名人和KOL的推动更会使粉丝对NFT的关注程度进一步加强。可以很清晰的看出，共识在市场上的力量非常强大，所有人都参与同一个游戏时，就会造就强大的力量。

    Yacht Club创始人认为，这种社区的参与感和从属感在互联网上一直是缺失的，而NFT可以帮助填补这一空白。Bored Apes可以成为人们的数字身份，它不仅仅是是一种收藏品，不挂在墙上或摆在架子上，而是呈现在你的头像图片上，是用户自我的象征。

    在Discord社区中，Bored Ape游艇俱乐部的Discord服务器上活跃着包括粉丝和NFT所有者，社交网络的讨论热度和相互社交，在更广泛的互联网中形成了一种俱乐部成员之间的隐形联系。

    我认为，无聊猿火的原因在于其打通了web3与web2的连接。购买的无聊猿NFT可以用作web2社交软件上的头像，可以体现出一种身份的象征（因为大家都知道这个价格贵）。这和戴表、开好车没有本质上的区别，不过是将奢侈品虚拟化的行为。不过防伪应该更加简单，直接查询NFT对应的公匙就好了。

    # NFT商业项目的特征

    CryptoPunks和Bored Ape这两个NFT项目都是完全封闭的系统，NFT发行数量是固定的（限制数量），创始人遵循承诺，在最初有限的发布数量之外，没有任何任何新增的NFT。

    # Bored Ape的特点

    在Bored Ape项目推出以前，大多数项目都倾向于低分辨率和像素化图像。如果单拿画风来说，和无聊猿精细的画风相比，像素画还是略显普通。Bored Ape的图像更为精美，而Yacht Club理念的来源是，创始人深受八十年代硬核、朋克摇滚、九十年代嘻哈的感染，是一种酒吧文化的代表，从网站上的世界末日酒吧场景，再到猴儿们本身的搞笑的风格。

    无聊猿游艇俱乐部，更像是一个游戏3A大作，而不仅仅是一个孤立的NFT组合。精细复杂的视觉效果、包含亚文化的时尚感和文学性质，使无聊猿的宇宙成为某种隐秘小众却充满吸引力的文化体现。

    Gargamel和Gordon Goner拉了两名程序员入伙，他们的名字是No Sass和Emperor Tomato Ketchup，主要负责区块链方面的工程工作。聘请插画师的投入占了前期大部分费用（约4万美元）。与许多NFT头像类型的俱乐部一样，卡通猿们随后被输入到一个算法程序中，该程序可以随机生成数千张具有独特的身体、头部、帽子和衣服组合的图像，就像数字装扮的芭比娃娃。

    某些独有特征：特殊毛发、激光眼、长袍，只会随机出现在少数人身上，使得拥有这些稀有款特征的猿猴更受欢迎，因此也会在二级市场更有价值。

    每张图片在最初的收藏者付款之前都是隐藏的，所以买一张图片有点像玩老虎机，或者是抽盲盒。如果得到一只隐藏款猿猴，就可以通过再次倒卖而获得巨大的利润。某些情况下，部分加密爱好者每人会购买数百个NFT。然后在价格上涨时卖掉他们的囤积NFT，但是必须保证有新的收藏者，之前的收藏者才能获利。

    # 人们在买什么

    就在发布后的第二天，一万张 "Bored App "全部售罄。2021年5月3日，OpenSea上的猿猴就涨到了1700美元。很多人害怕错过发售，这是对失去机会和错过的存在于人性中的恐惧。

    每个NFT其实都是一种独特象征。通常代表有门槛的在线社区、某个股票研究小组，或者是艺术鉴赏协会。当某只NFT被高价买走时，这套NFT中所有其他的的价值感知就会上升，就像一幅画如果在拍卖会上拍出创纪录的价格，就会增加该艺术家所有作品的价值一样。当买家把社交网络的头像变成新的NFT图片时，就代表着忠诚，也是一个信号——俱乐部的其他买家，快来关注呀。

    但稀有并不是人们为猿猴支付数百万美元的唯一原因。除了拥有独特的化身外，人们还可以通过这种NFT获得独家会员俱乐部的入场券。这让他们进入了精英的核心圈子，为他们带来了地位和更多的盈利机会。

    有用户认为，"当时没有人买得起CryptoPunk，"。这样一来，Bored Ape似乎是下一个最好的购买对象，它可以当做一个很酷的头像，价格也相对合理"。

    Artamonovskaja在Bored Ape推出后不久，就以大约1500美元的价格买下了一只猿，而现在同样的猴儿（戴着无聊猿游艇俱乐部品牌的棒球帽，有一种流行朋克的感觉）售价则是购入价格的十倍

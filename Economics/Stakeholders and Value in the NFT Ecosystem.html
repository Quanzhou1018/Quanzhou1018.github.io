    NFT生态系统中的角色们

    vuePress-theme-reco    2022

    NFT生态系统中的角色们

    NFTWeb3

    # 摘要

    NFT的争议性：支持者认为NFT的所有权开放、安全、不变，具有革命性的经济和文化潜力，而反对者认为NFT在环境问题和生态系统上产生了大量的不当行为。

    作者认为这一现象与人机交互有关，并且预示着未来交互产品的重要发展。

    本文贡献：为了更好地理解NFT现象，并为未来的HCI研究和设计提供信息，本文调查了NFT生态系统中的利益相关者及其之间的关系。基于从社会新闻网站Hacker news挖掘的开放数据，本文贡献了NFT生态系统中第一个数据支持的利益相关者模型。该模型揭示了创造者、所有者和技术人员的观点的细微差别;将投资公司和拍卖行视为知识和价值的仲裁者;并对未来的研究提出了建议。

    # 引言

    NFT被柯林斯词典选为2021年的年度词汇。NFT2014年被概念化，2018年实现标准化，到2021年因为其高昂的拍卖价格而闻名。

    # 市场规模大

    2021年NFTs的市场规模在120亿美元至260亿美元之间——比2020年的约2.5亿美元出现爆炸性增长。与其他hci相关市场相比较，目前估计数码相机的年市场规模约为180亿美元，微控制器的年市场规模为200亿美元，美国平面设计服务的年市场规模为130亿美元。

    # 从技术上讲

    NFT是分布式区块链数据库中注册所有权关系的一个data item。NFTs由智能合约管理，即存储在区块链上的计算机程序。目前超过97%的NFT智能合约位于以太坊区块链上，并遵循ERC-721技术标准。

    # 从内容上讲

    NFTs可以与一段数字媒体(如图像、视频、音乐)相关联，但也可以与物理财产(如房地产、艺术品、土地)、贷款、门禁钥匙、门票、游戏物品等相关联。由于区块链数据是开放的、不可变的、分布式的和加密安全的，NFTs提供了可靠的所有权记录;但目前NFTs并不存储任何媒介——它们只是提供了资产或媒介的链接，这些资产或媒体可能是安全的，也可能不是不可变的。

    # NFT到底是不是骗局

    很多人认为2021年NFT市场是投机泡沫（的确，从第一条Twitter这一NFT价值暴跌就能有所表现）。观点有：1、不可持续的爆炸性增长；2、难以辨认的价值主张；3、技术问题(如能源成本)；4、在NFT交易市场中欺诈、剽窃、操纵和错误信息的显著比例。

    而支持者在不断对NFT的生态系统进行重大投资，他们认为，由于潜在技术的承诺和NFT人工产品的文化价值，非NFT将在长期内变得越来越重要。的确，NFTs都是人机交互感兴趣的一种现象，与现代人机交互主题密切相关，并预示着交互产品的技术、经济和用户体验的转变。因此，研究它是很有必要的。

    # 本文干嘛了

    本文研究方向：本文主要通过以人为中心的研究来理解NFT的利益相关者（这样的文献很少），其可以对人机交互设计提供参考。 本文研究问题：谁是NFT生态系统中的当前利益相关者，他们的动机是什么，他们采取了什么行动? 本文贡献：构造了一个模型，这是NFT生态系统中第一个有数据支持的利益相关者关系的映射。

    # 方法和数据

    # 在哪获得的数据与知识

    为了收集有关nft和利益攸关方的知识，在2021年期间，我们跟踪了各种信息流。这包括但不限于在Twitter和Medium上追随思想领袖和杰出的生态系统行动者，加入Reddit和Discord上的相关社区，以及消费相关的YouTube视频和播客集。由于这门学科的迅速发展，这种方法是必要的，也是合适的——我们通过这种方式获得的知识在其他地方是无法获得的。

    此后，本文通过社会新闻网站Hacker News挖掘到的数据集进行内容分析，使用“NFT”搜索词搜索2021年期间的帖子。HN有大量关于技术和商业主题的内容，是我们研究问题的合适数据源，Barik等人此前进行的hci相关研究为其使用开创了先例。

    # 构建模型的方法

    大多数帖子链接到非站点内容，例如，在线文章或博客、社交媒体帖子、开源代码库或公司网站。我们遵循这些链接，并检查每个链接的内容，以迭代地构建并细化我们遇到的不同群体。对于每一篇文章，我们都在内容中记录了对利益相关者的直接引用和间接引用。例如，如果一篇帖子提到一场成功的拍卖，这就意味着有买家参与其中;或者，如果这篇文章是一篇代码仓库，这意味着开发人员也参与其中。基于此，我们引用内容来构建每个群体的描述，并映射它们之间的关系。

    # NFT生态系统里有哪些群体

    6种重要的利益相关者类别，这种分类主要是基于他们共同的动机和行为。

    艺术家和创作者

    NFT所有者

    开发者

    初创公司和投资者

    拍卖行

    企业

    潜在的，但由于他们与生态系统的接触较少，忽略掉的群体:艺术顾问、评论家、主流媒体、政府、学术研究人员，以及作品或肖像被利用的知识产权所有者。

    # 与另一已有模型Wilson(2021)的比较

    模型比较

    他们：将内容所有者(IP所有者)和个体创造者确定为NFT人工产品的发起者。 我们：在我们的数据中，我们发现在nft的“供应端”，艺术家和非艺术家创作者、开发者、公司、初创公司和投资者之间存在更复杂的相互作用，其中每个类别都有一个议程，在群体中很大程度上具有凝聚力，并且与其他类别截然不同。

    他们：确定了一种有点复杂的中介结构，例如，维护基础设施的以太坊基金会、加密货币交易所和创建nft的初创公司(如Dapper Labs)。 我们：没有发现这些行动者作为一个有凝聚力的中介类别的代表;并将它们归类为开发者，初创公司和创造者。此外，我们发现拍卖行（不包括在威尔逊等人的模型中）一直扮演着重要的中介角色，具有非常重要的后果行为。

    他们：将NFT所有者细分为投资者、收藏家、消费者和投机者。 我们：在我们的数据中，我们无法找到足够的细微差别来区分这些子类别，我们只是将生态系统的“需求方”作为所有者来捕捉。

    # 第一类：艺术家与创作者

    在我们的数据中，艺术家和创作者是目前为止最突出的利益相关者类别，仅次于NFT所有者。然而，与NFT所有者相比，美工和创作者的动机和行为得到了更好的定义，因为他们的个人资料、采访和故事非常丰富。

    我们将艺术家和非艺术家创作者区分为两个不同的群体，就他们对非艺术作品买家的价值主张而言:艺术家创造视觉艺术，并销售与艺术作品直接相关的非艺术作品。创作者在其他领域(如体育、科学、工程、喜剧)进行表演，并为NFTs营销资金。后者的例子包括加州大学伯克利分校对基于研究的NFTs的拍卖、meme创造者和非营利组织。

    艺术家和创作者经常相互合作。因此，这两个类别之间存在着重要的交集、关系和渗透性。

    艺术家和创作者的分类：

    原生媒介，作为NFT先驱的数字艺术家

    植根于其他媒体，抓住了NFT机会的艺术家

    专注于物理媒体，通过NFT增加价值主张的艺术家

    在艺术世界中没有明显血统的非正统派创作者

    为了利润或慈善与NFTs联系的名人

    将其IP代币化的互联网知名IP持有者

    在初创公司和公司找到工作的艺术家和设计师

    推销NFT来赚钱的科学家、非营利组织

    NFT现象对艺术家和创作者来说是好事。

    经济和文化趋势从根本上扩大了对视觉创意工作的需求，从而产生了意识、需求和工作。

    nft可以永久追踪所有权，并被设定为每当二级市场销售时为原创者赚取版税——这是一个新颖而重要的机会。

    NFTs是一种扩展“创造者经济”的新渠道，即使创造性工作容易被消费者补偿的技术可能性。

    因此，Adobe等公司正在为设计师和艺术家构建NFT功能，这说明了人机交互研究可以服务于这些利益相关者的一个方向。

    # 第二类：NFT所有者

    奇怪的是，NFT所有者很少发声。

    他们是最显著的利益相关者类别:许多帖子讲述了NFT的销售和拍卖，说明在这个帖子的参与用户中存在买家。然而，直接代表是非常罕见的，只有少数的采访和报告转向买家作为对象。

    除了投机交易，还有什么合法的价值主张会导致NFT的购买?通过研究，我们发现了以下NFTs的价值主张:

    游戏和虚拟世界将NFT作为功能性物品来执行，将玩法与NFT所有权结合起来。行业研究表明，玩家对具有货币价值的虚拟物品感兴趣，也对可能不是区块链原生的游戏感兴趣。

    将NFT作为一种线下活动的门票。在我们的研究中，我们没有遇到任何使用nft访问物理空间或服务的例子。然而，在与业内人士的私下交谈中，我们发现了在行业活动中使用这种方法的轶事。

    将NFT作为虚拟空间、社区和俱乐部的门票

    象征财富、创业和内部群体成员资格的标志（like PFP)

    NFTs反映了收藏家的审美和文化品味，从而形成了联系和社区。近两年由于疫情，很难在“现实生活”中建立基于文化和品味的当地关系。在线展示和交易NFT集合是一种替代方法，可以在全球范围内找到这种关系。

    NFTs被用作支付系统，而不是支票和信用卡，以补偿创造者。由于交易成本低、方便和/或意识形态，加密货币所有者更喜欢这种方式。（价值稳定性是否可以保证）

    去中心化自治组织(DAOs)和其他创新机构正在出现，他们会基于NFTs构建共同利益和治理机制

    尽管有大量潜在的NFT用例，但值得注意的是，在我们的数据中，像上面这样清晰的价值主张的真实例子很少。目前为止，我们所提到的主要项目以及投机交易占据了主导地位。与推测相比，由用例驱动的市场比例是多少?这是一个值得未来研究的问题，需要通过市场数据和以人为本的研究来回答。

    # 第三类：初创公司和投资者

    这类利益相关者拥有自己的生态系统，主要由创始人主导且专注于软件的私营公司和风险投资家（a16z、YC）组成。

    他们是一个高度互联的网络，与开发人员、创造者和公司融为一体;很难将网络中的个人划分为投资者和运营商——角色是动态的，网络中的许多人与其他参与者有着重大的经济利益关系。由于大量的投资关系和随之而来的信托责任，谨慎地说，初创公司和投资者是受财务驱动的。

    然而，他们的财务动机可以与将世界转变为理想状态的真实愿景相协同。初创公司在生态系统中具有重要意义，因为他们构建了基础设施和NFT——实际上，2021年所有的NFT交易都发生在OpenSea和其他由初创公司构建的市场上，绝大多数价值都积累在由初创公司构建的NFT项目中。

    投资者之所以重要，不仅是因为他们提供了使这些成为可能的资本，还因为他们通过思想领导对文化和叙事具有重大影响。例如，a16z是40多家区块链初创公司的直接投资者，包括coinbase、Opensea、Dapper Labs等；YC维护我们数据的来源HN。同时，大量关于NFTs的知识和叙述是由VC相关的个体撰写的。看NFT的VC从业者是高度活跃的内容创造者，解释和推广定义其参与者和支持者眼中的生态系统的技术和经济愿景。

    这些利益相关者对NFT生态系统具有重大影响，同时影响其技术和叙事。特别是风投，他们似乎有办法通过可信和多产的思想领导，在NFT项目中推动重大的财务价值。

    # 第四类：开发者

    开发者被划分为不同于创造者和初创公司的类别，因为他们具有两个特征:

    许多人承接项目时并没有与其他利益相关者有明显的个人联系(除了通过技术基础设施)

    他们所呈现的前景与那些有财务动机的初创公司、投资者、艺术家和创造者不同。

    虽然之前所提到的供应方分享了积极的前景，直言不讳地谈论了NFTs的潜力和价值，但是许多开发者则会提出或代表批评、怀疑和清醒的分析。（不会过多考虑经济利益）

    有趣的是，除了黑客事件以及初创公司和公司之外，其他媒体中相对较少提及开发者。在以技术为中心的生态系统中，这是一个有点令人担忧的特性。

    # 第五类：拍卖行

    拍卖行在市场中扮演着极具影响力的角色。尽管如此，在我们的数据叙述中，拍卖行的作用并不突出，在相关研究中也不存在。拍卖行在NFT生态系统中的作用需要全新的研究，因为它是这一现象的经济学、文化和社会学的核心。

    我们数据中的显著特征是，NFTs的主题直到2021年2月底才在HN得到重视，这与佳士得拍卖行高调宣布Beeple卖出了6900万美元相吻合。在2012-2021年期间，913篇匹配“NFT”的HN帖子中，只有15篇在该事件之前。考虑到HN社区主要由业务和技术的早期采用者组成，这样的事应当被提前预知。

    根据我们的数据和我们对艺术品市场的结构和权力关系的初步研究，NFT市场的财务价值中有很大一部分来自佳士得和苏富比等拍卖行的参与。这些拍卖行作为合法性的权威和仲裁者，对艺术品的定价有着强烈的直接影响，并通过丰富的销售和营销策略对市场的需求方产生间接影响。

    目前艺术领域的NFT研究主要集中在潜力和美学，而定量研究则忽略了拍卖行、风险投资等权威机构的作用。刻画拍卖行对非主流金融市场的影响是金融、经济学、社会学和艺术交叉领域未来工作的一个有趣话题。

    # 第六类：企业

    在这个类别中，我们研究了专注于非NFT业务的成熟公司，包括像耐克、EA、Epic等游戏公司;大型科技行业公司，如Shopify和Meta;媒体公司。

    这些参与者在生态系统中的参与在很大程度上是保守的，取决于他们所立足的行业。模式包括:

    初创后的软件公司试图利用区块链趋势，包括NFTs，通过广泛的品牌战略转变（meta meta meta！)

    与游戏和创意相关的公司果断地整合或远离NFT

    非软件公司通过投资或收购来接受NFT

    被定位为受到创造者经济影响的大型新闻媒体企业

    在生态系统中，大多数公司是追随者而不是领导者，可能被认为是在HCI的范围之外。然而，他们之所以重要，有两个原因：

    他们与NFTs的接触被公众视为市场整体合法性的标志，潜在地推动了价值。

    大公司的行动会引发公众的反应，而公众的反应又会体现在我们的数据中（作为一个话题引爆机haha）——例如，GSC Game World宣布又取消了在即将到来的比赛中使用NFTs的计划，考虑到球迷的能源消耗和财务风险。

    # 结论

    我们提出了NFT生态系统的利益相关者模型，该模型在2021年推动了一种重要的技术、文化和经济现象。据我们所知，这是第一个由开放数据支持的模型，并推进了之前数据和方法不透明的工作。

    正如我们在上面提出的模型，我们也强调了对未来HCI(人机交互）和跨学科研究的影响和建议。我们研究结果的亮点包括:

    对NFT艺术家/创作者和所有者的动机和价值主张进行了细致入微的、有数据支持的研究，这可以为HCI提供信息。

    将风投公司定位为拥有既得利益的知识创造者，将拍卖行定位为价值仲裁者，这需要进一步研究。

    令人惊讶的是，在我们的数据中，开发者和普通公众的比例很低。

    我们研究的一个局限性是我们的数据相对有限，只基于一个社会新闻网站。因此，我们的模型应该被作为一个可能的角度来看待这个问题，而不是一个结论性的解释现象。

    未来的工作可以在不同的数据集上研究类似的问题，以建立更具包容性的模型，并执行更高级的分析，也可以涉及定量特征。

    对于NFTs的价值、合法性和实用性，目前还没有定论。一方面，它为创意、非营利组织和创新者提供了令人兴奋的机会。另一方面，像维护区块链的高能源成本和生态系统中的错误行为这样的担忧也有好处。我们希望像我们这样的研究可以阐明这两个方面，最终克服缺点，实现技术的积极潜力。在不久的将来，我们期待新的技术标准能够取代当前基于erc721的NFTs，并将其应用到视觉艺术之外。
